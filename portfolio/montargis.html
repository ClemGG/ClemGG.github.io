SHOWROOM INDUSTRIE MONTARGIS

Une simulation Web représentant le fonctionnement d'une chaîne de production
dans une usine de robotique à Montargis.

DESCRIPTION

Développement :

Date de sortie : Juillet 2024
Moteur : Unity3D
Lien : Github

  Le Showroom Industrie 4.0 Equans Digital est une simulation web représentant la chaîne de production d'une usine robotique à Montargis, France.

  Le site est accessible dans de multiples langues, et offre une visite guidée de la chaîne de production avec des points d'intérêt cliquables dans l'espace.

  Chaque point d'intérêt offre une description détaillée du maillon de la chaîne correspondant, ainsi qu'une vidéo démonstrative de son cas d'usage.
